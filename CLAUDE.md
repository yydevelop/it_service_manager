## Claude Code設定

### パーミッション設定
Claude Codeでパーミッションチェックをスキップする方法：

1. **コマンドラインオプション**:
   ```powershell
   claude --permission-mode bypassPermissions
   ```
   または
   ```powershell
   claude --dangerously-skip-permissions
   ```

2. **推奨される方法（安全）**:
   ```powershell
   claude --permission-mode dontAsk
   ```
   パーミッションを尋ねずに自動的に承認します。

3. **設定ファイルでの永続化**:
   `.claude-code/settings.json`に以下を追加：
   ```json
   {
     "permissionMode": "bypassPermissions"
   }
   ```

**注意**: `--dangerously-skip-permissions`はインターネットアクセスのないサンドボックス環境でのみ推奨されます。
